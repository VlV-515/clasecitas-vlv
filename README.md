# clasecitas-vlv

Small utility-first CSS toolkit with Tailwind-like class names and a Bootstrap 5-style grid.

This package ships plain CSS for any project and SCSS source for teams that want to customize or inspect the source.

## Install

```bash
npm install clasecitas-vlv
```

## Usage

```js
import "clasecitas-vlv/css";
```

```scss
@use "clasecitas-vlv/scss";
```

Legacy aliases are optional and not included by default:

```js
import "clasecitas-vlv/legacy";
```
